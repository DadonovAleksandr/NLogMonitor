# Assets

Эта папка содержит ресурсы приложения (иконки, изображения и т.д.).

## Иконки

### log.png
Главная иконка приложения (256×256 пикселей), скопированная из проекта nLogViewer.

**Используется в:**
- Web-приложение: `client/public/log.png` (favicon)
- Desktop приложение: конвертируется в `src/nLogMonitor.Desktop/icon.ico`

### Обновление иконки

Если нужно обновить иконку приложения:

1. Замените `log.png` на новое изображение (рекомендуется 256×256 пикселей)
2. Запустите скрипт конвертации:
   ```powershell
   cd assets
   .\convert-icon.ps1
   ```
3. Скопируйте обновлённую иконку в public:
   ```bash
   cp log.png ../client/public/log.png
   ```

## Скрипты

### convert-icon.ps1
PowerShell скрипт для конвертации PNG в ICO формат с поддержкой множественных размеров (256, 128, 64, 48, 32, 16 пикселей).
